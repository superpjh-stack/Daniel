# Plan: 달란트 시장 쿠팡 스타일 개선

## 메타데이터
- **기능명**: shop-coupang-style
- **우선순위**: High
- **예상 규모**: Medium
- **작성일**: 2026-02-18

---

## 1. 배경 및 목적

### 현황
- `/shop` 달란트 시장 페이지가 단순 그리드 카드 형태로 구현됨
- 상품 이미지 없이 텍스트만 표시 (실제 이미지 없음)
- 가격(달란트) 수정 시 전체 편집 모달을 열어야 함 (번거로움)
- 기본 샘플 상품 6개만 있고 이미지/구체적 정보 없음

### 목적
- 쿠팡 스타일의 직관적이고 시각적으로 매력적인 쇼핑몰 UI로 개선
- 실제 상품 이미지(공개 CDN/Unsplash)와 구체적인 제목 추가
- 달란트(가격) 인라인 빠른 수정으로 관리 편의성 향상
- 초등학생이 쉽게 사용할 수 있는 쿠팡 같은 UX 제공

---

## 2. 기능 요구사항

### FR-01: 쿠팡 스타일 상품 카드 UI 개선
- **이미지 영역**: 4:3 비율 또는 정사각형, 크게 표시
- **상품명**: 2줄 말줄임, 굵은 폰트, 쿠팡처럼 구체적 제목
- **별점 표시**: ⭐ 4.5~5.0 (고정값, 시각적 요소)
- **리뷰 수**: "(12명 구매)" 형태 뱃지
- **가격 표시**: 달란트 아이콘 + 숫자 크게 + 볼드
- **뱃지**: "NEW", "SOLD OUT", "인기" 등 쿠팡 스타일 뱃지
- **구매하기 버튼**: 풀 너비, 오렌지/쿠팡 컬러 스타일

### FR-02: 달란트(가격) 인라인 빠른 수정
- 관리자/교사 모드에서 상품 카드의 가격 영역에 편집 아이콘 표시
- 가격 클릭 → 인풋 필드로 전환 → 엔터/저장 버튼으로 즉시 저장
- 저장 후 카드 UI 즉시 업데이트 (리로드 없이)
- 기존 전체 편집 모달(수정 버튼)은 유지

### FR-03: 실제 상품 시드 데이터 추가
쿠팡에서 실제 판매되는 초등부 어린이 상품 카테고리별 10~15개:

| 카테고리 | 상품 예시 | 이미지 |
|---------|---------|--------|
| 학용품 | 스테들러 연필 세트, 모나미 볼펜, 색연필 세트 | Unsplash 공개 이미지 |
| 간식 | 포켓몬 빵, 초코파이, 젤리 세트 | Unsplash 공개 이미지 |
| 문화 | 도서상품권, 영화 관람권 | Unsplash 공개 이미지 |
| 특별 | 치킨 쿠폰, 피자 1판 쿠폰 | Unsplash 공개 이미지 |
| 기타 | 보드게임, 레고 미니 세트 | Unsplash 공개 이미지 |

### FR-04: 상품 추가 모달 개선
- 이미지 URL 입력 시 미리보기 (기존 기능 유지)
- **추천 상품 템플릿** 버튼: 미리 준비된 상품 목록에서 1클릭 선택
- 달란트 가격 슬라이더 or 빠른 선택 버튼 (5 / 10 / 20 / 30 / 50)

### FR-05: 모바일 최적화 그리드
- 모바일: 2열 그리드 (현재와 동일)
- 태블릿: 3열
- 데스크톱: 4~5열
- 상품 카드 내 텍스트 크기 모바일 최적화

## 3. 쿠팡 스타일 UI 레퍼런스

```
┌─────────────────────┐
│  [이미지 4:3]       │ ← 크고 선명한 상품 이미지
│  [NEW] [인기]       │ ← 뱃지 오버레이
├─────────────────────┤
│ 스테들러 노리스     │ ← 상품명 2줄
│ 연필 12자루 세트    │
│ ⭐⭐⭐⭐⭐ (4.8)   │ ← 별점
│ 👥 23명 구매       │ ← 구매자 수
├─────────────────────┤
│ ⭐ 25 달란트       │ ← 가격 (관리자: ✏️ 클릭수정)
│                     │
│ [  구매하기  ]      │ ← CTA 버튼
└─────────────────────┘
```

## 4. 구현 범위

### 변경 파일
- `src/app/(dashboard)/shop/page.tsx` — UI 전면 개선 (메인 작업)
- `prisma/seed.ts` — 상품 시드 데이터 업그레이드 (이미지 포함)

### 신규 파일 없음
- 기존 API(`/api/shop/products`, `/api/shop/products/[id]`) 그대로 활용
- Prisma schema 변경 없음 (`image` 필드 이미 존재)

## 5. 이미지 전략

**Unsplash Source API** 사용 (무료, 라이선스 없음):
- 형식: `https://source.unsplash.com/400x400/?{keyword}`
- 예: `https://source.unsplash.com/400x400/?pencil,stationery`
- 또는 특정 이미지 ID 사용으로 안정적 표시

**대안**: Lorem Picsum (`https://picsum.photos/seed/{id}/400/400`)

## 6. 달란트 인라인 수정 플로우

```
[관리자 모드]
가격 표시: "⭐ 25 달란트 [✏️]"
               ↓ 클릭
가격 편집: "[_25_] 달란트 [✓] [✗]"
               ↓ 엔터 or ✓
API PUT /api/shop/products/{id} → price 업데이트
               ↓ 성공
가격 즉시 업데이트 (로컬 state 반영)
```
