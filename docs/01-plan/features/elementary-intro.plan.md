# Plan: elementary-intro (초등부 소개 메뉴)

## 개요
- **기능명**: elementary-intro
- **작성일**: 2026-02-19
- **우선순위**: Medium
- **예상 규모**: Small (1~2일)

## 배경 및 목적

동은교회 초등부 앱(다니엘)에 **초등부 소개 페이지**를 추가한다.
현재 사이드바에는 출석, 달란트, 공지사항 등 운영 메뉴만 존재하며,
부서 정체성(비전·교육목표·교육방침·담당자)을 한눈에 보여주는 소개 페이지가 없다.

참고: https://www.dongeun.or.kr 초등부 소개 페이지 (이미지 슬라이드 형식)

## 요구사항

### 기능 요구사항
| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-01 | 사이드바에 "초등부 소개" 메뉴 항목 추가 | Must |
| FR-02 | `/intro` 경로에 소개 페이지 생성 | Must |
| FR-03 | 소개 페이지: 부서 비전·슬로건 표시 | Must |
| FR-04 | 소개 페이지: 교육 목표 섹션 | Must |
| FR-05 | 소개 페이지: 교육 방침 섹션 | Must |
| FR-06 | 소개 페이지: 담당 교역자/교사 소개 섹션 | Should |
| FR-07 | 관리자가 소개 내용(텍스트)을 편집 가능 (Setting 모델 활용) | Should |
| FR-08 | 모든 역할(admin/teacher/parent)에게 표시 | Must |

### 비기능 요구사항
- 모바일 우선 반응형 디자인
- 기존 글래스모피즘/그라데이션 디자인 시스템 일관성 유지
- 페이지 로딩 없이 정적 콘텐츠로 빠른 표시

## 페이지 구성 (초안)

```
/intro
├── 히어로 배너 (부서 이름 + 슬로건)
├── 비전 카드 (주요 비전 문구)
├── 교육 목표 섹션 (3~4개 목표)
├── 교육 방침 섹션 (핵심 방침들)
└── 담당자 소개 (교역자 + 교사진)
```

## 콘텐츠 계획

### 기본 콘텐츠 (하드코딩 초기값)
동은교회 초등부 소개 페이지의 이미지 슬라이드 내용을 기반으로 구성:
- **슬로건**: "말씀으로 자라는 하나님의 자녀들"
- **비전**: 하나님 말씀 위에 굳게 서는 아이들
- **교육 목표**: 예배, 말씀, 전도, 섬김
- **교육 방침**: 주일학교 중심 교육, 가정과의 연계

> 관리자가 Settings 테이블을 통해 내용 수정 가능하도록 설계

## 기술 구현 방향

### 1단계: 정적 소개 페이지 (MVP)
- `src/app/(dashboard)/intro/page.tsx` 생성 (서버 컴포넌트)
- 콘텐츠는 하드코딩 (빠른 구현)
- 사이드바에 `BookOpen` 아이콘 + "초등부 소개" 추가

### 2단계: 관리자 편집 기능 (선택)
- `Setting` 모델에 소개 콘텐츠 키-값 저장
- `/api/intro` API 라우트로 GET/PUT
- 관리자 편집 UI (`/settings` 페이지 확장 또는 별도 `/intro/manage`)
